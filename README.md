# flickr-wallpaper
Gnome-Shell extension for changing wallpaper with random Flickr image

## Dependencies
* curl

## Install
```
cd $HOME/.local/share/gnome-shell/extensions
git clone https://github.com/pagondel/flickr-wallpaper.git
```
